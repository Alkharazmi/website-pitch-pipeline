> Deep research for the Local Business Website Pitch Pipeline.

## Method

| Signal | Why it matters |
|--------|----------------|
| skills.sh **install volume** | Real agent adoption, not just stars |
| **Maintainer** | Anthropic, Vercel, domain experts (Bakaus, Haines, Kowalski) |
| **Scope fit** | Landing/demo UI + conversion/outreach writing (this pipeline) |
| **Overlap policy** | Prefer 1 primary + 1 specialist per job; avoid 10 skills fighting |

**Not “best” solely by stars.** A 40k-star mega-pack can be noisier than a focused 150k-install skill.

## Recommended install commands (new machine)

```bash
# --- Design (Tier S) ---
npx skills add anthropics/skills --skill frontend-design
npx impeccable install
# or: npx skills add pbakaus/impeccable
npx skills add vercel-labs/agent-skills --skill web-design-guidelines
npx skills add leonxlnx/taste-skill
# includes design-taste-frontend, high-end-visual-design, minimalist-ui,
# industrial-brutalist-ui, redesign-existing-projects, brandkit, imagegen-*, etc.
npx skills add nextlevelbuilder/ui-ux-pro-max-skill
npx skills@latest add emilkowalski/skills

# --- Design (Tier A extras) ---
npx skills add mattpocock/skills --skill design-an-interface edit-article
npx skills add arvindrk/extract-design-system

# --- Copy / writing (Tier S + A) ---
npx skills add coreyhaines31/marketingskills
# core: copywriting copy-editing cold-email page-cro cro marketing-psychology
# also: seo-audit content-strategy emails ad-creative site-architecture ...
npx skills add hardikpandya/stop-slop
npx skills add petergyang/no-ai-slop   # or clone skill
npx skills add boraoztunc/skills --skill ogilvy
# or full ogilvy if published separately
npx skills add content-designer/ux-writing-skill
```

If `leonxlnx/taste-skill` package name fails, skills are often already present as individual folders (`design-taste-frontend`, etc.) from prior installs — verify with `npx skills list`.

## How to stack them (do not run 15 at once)

### Planning a unique site (Loop 2)

```text
1. ADHD          → divergent concepts (avoid early lock-in)
2. frontend-design + design-taste-frontend → aesthetic direction
3. Pick ONE lane: high-end | minimalist | industrial-brutalist | brandkit
4. impeccable init / DESIGN.md
5. copywriting + marketing-psychology + ogilvy → messaging
6. page-cro → persuasion structure
7. stop-slop + no-ai-slop → clean prose in brief
8. emil-design-eng → motion plan if needed
```

### Building (Loop 3)

```text
1. implementation.md is law
2. impeccable + frontend-design while coding
3. emil review-animations if motion
4. web-design-guidelines / vercel patterns for Next quality
5. copy-editing + stop-slop + no-ai-slop on all UI strings
6. cold-email + stop-slop for EMAIL_DRAFT.md
7. ux-writing-skill for microcopy
```

### Anti-patterns

- Loading **impeccable + 4 taste skills + ui-ux-pro-max** with conflicting palettes → muddy output.  
  **Pick one visual lane**, keep impeccable + frontend-design as governors.  
  **ui-ux-pro-max** is a toolbox: open it when you need palette/type/product-type suggestions; do not let it override an already chosen lane.  
- Using only SEO skills for outreach email → sounds robotic. Always anti-slop.  
- Skipping **page-cro** → pretty pages that don’t convert interest.
